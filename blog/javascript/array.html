Array 주요 매서드

이 글에 대하여

자바스크립트 배열은 데이터 집합을 다루기 위한 객체이며, 다양한 메서드를 제공해요.

#JavaScript#sort

요소 추가/제거 관련

스택 (LIFO)

  • push(elem)
    배열 끝에 추가하고, 길이를 반환해요.
  • pop()
    배열 끝에서 제거하고, 제거한 요소를 반환해요.

큐(FIFO)

  • unshift(elem)
    배열 앞에 추가하고, 길이 반환해요.
  • shift()
    배열 앞에서 제거하고, 제거한 요소 반환해요.

순회/탐색

forEach(callback)

  • 배열의 각 요소에 대해 함수를 실행해요. 반환값은 없어요.
1
[1, 2, 3].forEach(num => console.log(num)); // 1, 2, 3

map(callback)

  • 각 요소를 가공해서 새 배열을 반환해요.
1
const arr = [1, 2, 3];

2
const doubled = arr.map(num => num * 2);

3
console.log(doubled); // [2, 4, 6]

filter(callback)

  • 조건을 만족하는 요소만 모아 새 배열을 반환해요.
1
const arr = [1, 2, 3, 4];

2
const evens = arr.filter(num => num % 2 === 0);

3
console.log(evens); // [2, 4]

find(callback)

  • 조건을 만족하는 첫 번째 요소를 반환해요.
1
const arr = [5, 12, 8, 130, 44];

2
const found = arr.find(num => num > 10);

3
console.log(found); // 12

some(callback)

  • 조건 만족하는 요소가 하나라도 있으면 true를 반환해요.
1
const arr = [1, 2, 3];

2
console.log(arr.some(num => num > 2)); // true

every(callback)

  • 모든 요소가 조건을 만족해야 true를 반환해요.
1
const arr = [2, 4, 6];

2
console.log(arr.every(num => num % 2 === 0)); // true

누적/집계

reduce(callback, init)

  • 누적해서 값 하나를 반환해요.
1
[1, 2, 3].reduce((acc, cur) => acc + cur, 0); // 6

정렬/가공

sort(compareFn)

  • 배열을 정렬(원본 변경) 해요. 기본은 문자열 기준 정렬이고, 숫자는 비교 함수를 넣어야 해요.
    a - b 오름차순 정렬
    b - a 내림차순 정렬
1
[3, 1, 2].sort();                // ["1","2","3"] (문자열 기준)

2
[3, 1, 2].sort((a, b) => a - b); // [1, 2, 3]

reverse()`

  • 배열을 뒤집기(원본 변경) 해요.
1
[1, 2, 3].reverse(); // [3, 2, 1]

slice(start, end)

  • 배열의 일부를 복사해서 새 배열 반환해요.
1
const arr = [1, 2, 3, 4];

2
console.log(arr.slice(1, 3)); // [2, 3]

splice(start, count, ...items)

  • 배열에서 요소를 삭제·추가·교체(원본 변경) 해요.
1
const arr = [1, 2, 3, 4];

2
arr.splice(1, 2, 99); // 1번 인덱스부터 2개 삭제, 99 추가

3
console.log(arr); // [1, 99, 4]

concat(arr)

  • 배열 이어붙여 새 배열 반환해요.
1
[1, 2].concat([3, 4]); // [1, 2, 3, 4]

join(sep)

  • 배열 요소를 문자열로 합쳐요.
1
["a", "b", "c"].join("-"); // "a-b-c"

flat(depth)

  • 다차원 배열을 평탄화해요.
1
[1, [2, [3]]].flat();    // [1, 2, [3]]

2
[1, [2, [3]]].flat(2);   // [1, 2, 3]

localeCompare()

  • 문자열 A에서 A.localeCompare(B)를 하면
    음수 (예: -1) A가 B보다 앞이다. (즉 A < B)
    0 두 문자열이 같다.
    양수 (예: 1) A가 B보다 뒤다. (즉 A > B)
1
console.log("a".localeCompare("b")); // -1  ("a" < "b")

2
console.log("b".localeCompare("a")); // 1   ("b" > "a")

3
console.log("a".localeCompare("a")); // 0   ("a" == "a")
