게시 글 페이지

  • generateStaticParams 자세히
    빌드 시점에 어떤 동적 경로를 미리 생성할지 Next.js에 알려 주는 역할을 해요.
    예를 들어 app/blog/[slug]/page.tsx 같은 동적 세그먼트가 있을 때, 이 함수가 반환한 params 목록대로 각각의 HTML을 정적으로 생성해 두므로 페이지 로딩 속도가 빨라지고 서버 요청 없이도 콘텐츠를 제공할 수 있어요.

  • generateMetadata 자세히
    동적 경로에 맞춘 <head> 메타데이터를 서버 컴포넌트 차원에서 동적으로 생성하기 위해 사용돼요.
    빌드 시 고정된 export const metadata 대신, 이 함수 내에서 params를 통해 슬러그를 받고 외부 API나 파일에서 제목·설명 등을 가져와 반환함으로써, SEO 최적화 및 소셜 공유 태그를 페이지별로 맞춤 설정할 수 있어요.
