주소창에 google.com을 입력하면 일어나는 일

이 글에 대하여

브라우저가 URL 입력을 처리하고, 네트워크를 거쳐 화면에 그려지는 핵심 과정

#Frontend#Browser#Network#Rendering

1. URL 해석

  • 검색어인지, URL * 인지 판단해요.
  • google.com은 기본적으로 HTTPS로만 접속해요. (HSTS * 때문에 http는 차단돼요.)

2. DNS 조회

  • 도메인 네임 부분을 DNS * 서버에서 검색해요.
  • google.com 도메인의 IP 주소를 확인해요.

3. 서버 연결

  • QUIC(HTTP/3) * 또는 TCP(HTTP/2,1.1) *로 연결해요.
  • TLS 핸드셰이크 *로 암호화 및 인증서 검증

4. HTTP 요청/응답

  • 브라우저가 GET / 요청해요.
  • 서버가 HTML 응답을 전송해요.
