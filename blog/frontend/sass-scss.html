Sass(SCSS)

이 글에 대하여

Sass(SCSS)는 CSS를 더 편리하게 작성할 수 있게 해 주는 CSS 전처리기(Preprocessor)예요. 반복되는 코드나 복잡한 스타일을 효율적으로 관리할 수 있어요.

#Frontend#Sass#SCSS#CSS

특징

변수 사용 가능

  • 색상, 폰트 크기, 간격 등을 변수로 관리할 수 있어요.
  • 유지보수와 테마 변경이 쉬워져요.
1
$main-color: #3498db;

2


3
body {

4
  background: $main-color;

5
}

중첩(Nesting)

  • 선택자를 계층적으로 작성할 수 있어, 가독성이 높아져요.
1
nav {

2
  ul {

3
    li {

4
      a {

5
        color: blue;

6
      }

7
    }

8
  }

9
}

믹스인(Mixin)

  • 반복되는 코드를 함수처럼 재사용할 수 있어요.
1
@mixin flex-center {

2
  display: flex;

3
  justify-content: center;

4
  align-items: center;

5
}

6


7
.box {

8
  @include flex-center;

9
}

상속(Extend)

  • 공통 스타일을 상속해서 코드 중복을 줄여요.
1
.btn {

2
  padding: 10px;

3
  border-radius: 4px;

4
}

5


6

9
}
