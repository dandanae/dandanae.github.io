# JavaScript란?
## 정의
- **스크립트 언어**
  실행 시 해석(인터프리팅)되는 언어예요.
- **멀티 패러다임 언어**
  객체지향, 함수형, 명령형 프로그래밍 방식을 모두 지원해요.
- **동적 언어**
  실행 중에 변수 타입이 결정돼요.

## 주요 특징
- **인터프리터 언어**
  자바스크립트 코드는 엔진(V8 등)이 실행 시점에 바로 해석돼요.
- **동적 타입(Dynamically Typed)**
  변수는 실행 중에도 자유롭게 타입이 바뀔 수 있어요.
- **이벤트 기반(Event-driven)**
  DOM 이벤트, 사용자 입력, 네트워크 응답 같은 상황에 반응해 실행돼요.
- **싱글 스레드 (Single Thread)**
  하나의 스레드만 사용해 한 번에 하나의 작업만 실행해요.
- **논 블록킹 (Non-Blocking)**
  오래 걸리는 작업은 기다리지 않고, 비동기적으로 다른 코드부터 실행해요.
- **비동기 처리 지원**
  `callback`, `promise`, `async`/`await`으로 비동기 로직을 다룰 수 있어요.
- **독특한 실행 메커니즘**
  [호이스팅](/blog/frontend/hoisting), [클로저](/blog/frontend/closure), 렉시컬 스코프 같은 동작 원리를 갖고 있어요.

# 싱글 스레드와 논 블록킹
```js
console.log("A");

setTimeout(() => {
  console.log("B");
}, 1000);

console.log("C");
```

**출력 결과**
```plainText
A
C
B
```

싱글 스레드이지만, `setTimeout` 같은 비동기 함수 덕분에 논 **블록킹 방식으로 동작**해 다른 코드(`C`)를 먼저 실행해요.
