# 특징
## 변수 사용 가능
- 색상, 폰트 크기, 간격 등을 변수로 관리할 수 있어요.
- 유지보수와 테마 변경이 쉬워져요.

```scss
$main-color: #3498db;

body {
  background: $main-color;
}
```

## 중첩(Nesting)
- 선택자를 계층적으로 작성할 수 있어, 가독성이 높아져요.

```scss
nav {
  ul {
    li {
      a {
        color: blue;
      }
    }
  }
}
```

## 믹스인(Mixin)
- 반복되는 코드를 함수처럼 재사용할 수 있어요.

```scss
@mixin flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

.box {
  @include flex-center;
}
```

## 상속(Extend)
- 공통 스타일을 상속해서 코드 중복을 줄여요.

```scss
.btn {
  padding: 10px;
  border-radius: 4px;
}

.btn-primary {
  @extend .btn;
  background: blue;
}
```

## 모듈화(Partials & Import)
- 여러 개의 SCSS 파일로 나누고 `@import`나 `@use`로 불러올 수 있어, 대규모 프로젝트에 유리해요.
